Epitre is a modern web application that allows users to create, customize, and share interactive narrative cards through unique shareable links. Whether it's for a special occasion, a creative message, or an engaging story, Epitre provides an intuitive platform to craft beautiful, animated experiences.

## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone https://github.com/Missive-Project/epitre
cd epitre
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
